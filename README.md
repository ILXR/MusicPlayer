# kakaMusic
一款基于[MeetMusic](https://github.com/lijunyandev/MeetMusic)的Android本地音乐播放器，在原播放器的基础上添加了音乐播放bar以及修改了部分bug。同时加强了注释，尽力能够让项目作为很好的学习项目
