## 总需求

* 本地音乐播放器 (80%)
* WebSocket通信 (80%)
* 蓝牙通信 (50%)
* 控制音乐接口 (0%)
  * 状态：播放、暂停
  * 切歌：上一首、下一首
  * 音量：增加、减小
  * 列表：随机播放、单曲循环、顺序播放
