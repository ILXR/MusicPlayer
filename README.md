## 总需求

* 本地音乐播放器
* WebSocket通信
* 蓝牙通信
* 控制音乐接口
  * 状态：播放、暂停
  * 切歌：上一首、下一首
  * 音量：增加、减小
  * 列表：随机播放、单曲循环、顺序播放
